AWS Deep Racer Project

End of degree project · Vanier College · 2022

67th / 2,325Finish in the unranked division
Top 3%Of all entrants
Self hostedLinux training server

For this end of degree project I trained a reinforcement learning (RL) driving model for the AWS DeepRacer, a 1/18th scale autonomous car that learns to lap a track, and finished 67th of 2,325 entrants (top 3%) in the unranked division.

Unlike conventional programming, where you tell the system exactly what to do, an RL agent improves by trial and error against a reward function. The core engineering problem was designing that reward and the action space so the model would converge on a fast, stable racing line rather than getting stuck or driving erratically. I combined vehicle dynamics constraints with reward function tuning to shape that behaviour, and trained the model on a self hosted Linux server to avoid per hour cloud training costs, which also let me iterate on far more model variants than the budget would otherwise have allowed.
